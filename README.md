# Buscador de Pacientes

Sitio publico para la busqueda de pacientes del Hospital General "Dr. Miguel Perez Carreño" IVSS.

- `index.html`: pagina de busqueda (tambien se usa como snippet embebido en GHL).
- `pacientes.json`: datos de pacientes, se actualiza aqui para reflejar cambios sin tocar el codigo embebido.
